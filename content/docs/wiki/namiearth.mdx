# 🏰 NamiEarth 機能解説

NamiEarthは、実際の地球を模したマップで「金（ゴールド）」を基軸通貨とし、Townyによる国家運営を楽しむサーバーです。

<Info>
  このシステムは**NamiEarth (S3)**専用です。
</Info>

---

## 🗳️ 投票 (Vote)
サーバーに投票することで、金インゴットや便利なアイテムを獲得できます。
※NamiEarthにログインした状態で投票を行ってください。

1. **[JMS (日本マインクラフトサーバー)](https://minecraft.jp/servers/nami.mcsv.win)** (ログインが必要)

## 💰 GoldBank（経済システム）
金（ゴールド）は素材としてだけでなく、サーバー内の「通貨」として機能します。

| コマンド | 説明 |
| :--- | :--- |
| `/deposit <数>` | 手持ちの金インゴットを銀行に預けます。 |
| `/withdraw <数>` | 銀行から金インゴットをアイテムとして引き出します。 |
| `/bal` | 自分の銀行残高を確認します。 |
| `/bank top` | サーバー内の富豪ランキングを表示します。 |

---

## 🏗️ Towny（町と国家）
土地を保護し、仲間と発展させるためのシステムです。

| コマンド | 費用 | 説明 |
| :--- | :--- | :--- |
| `/t new <名>` | **64g** | 今立っている場所を中心に町を作成します。 |
| `/t deposit` | - | 町の銀行にお金を預けます（維持費に充てられます）。 |
| `/n new <名>` | **2048g** | 自分の町を首都として新しい国を建国します。 |

<Warning>
  **維持費の重要性**: 町の残高がなくなると、土地保護が消失し、町が崩壊します。こまめに `/t deposit` しましょう。
</Warning>

---

## 🧪 便利機能
- **経験値ボトル化 (`/bottle`)**: `/bottle store all` で所持している全経験値をボトル化して保管できます。
- **ランダムワープ (`/rtp`)**: `/rtp` で世界のランダムな地点へテレポートします。
- **PvP設定**: `/pvp off` で不意の戦闘を回避できます（中立地帯や戦争時を除く）。

---

## 💡 生き残るためのヒント
- **資産の守り方**: 貴重な金は常に `/deposit` で預け、経験値は `/bottle` でアイテム化してチェスト等に保管しましょう。
- **名称の制限**: 町名・国名に日本語は使えません。必ず半角英数字で命名してください。
- **国名ルール**: NamiEarthでは、国名は**実在または歴史上の国名**にする必要があります。
- **データの長期保管**: **30日以上ログインがない場合**、Townyの土地データは自動的に削除（解放）されます。
- **迷ったらWikiを確認**: [Townyガイド](/wiki/towny) や [攻城戦ガイド](/wiki/siegewar) を併せて読みましょう。
