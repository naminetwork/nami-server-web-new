# 🛒 QuickShop ガイド

QuickShopは、チェストを使って誰でも簡単に自分のお店（ショップ）を作成できるシステムです。
**NamiEarth** と **サバイバル鯖**（プロットエリア限定）で利用可能です。

<Info>
  ショップを作成するには、チェストと販売したいアイテムが必要です。お金の単位は **g** または **d** です。
</Info>

## ショップの作り方

自分のショップを作成する手順は以下の通りです：

1. **チェストを設置する**: ショップにしたい場所にチェストを置きます。
2. **アイテムを持ってクリック**: 販売したいアイテムを手に持ち、チェストを**左クリック**します。
3. **価格を入力**: チャット欄に「1個あたりの価格」を入力します。
   - 例: `10` と入力すると、1個10g（または10d）で販売されます。
4. **完成！**: チェストの上にアイテムが浮き、看板が自動的に設置されれば作成完了です。

<Warning title="注意">
  看板を設置するスペースがない場合は作成できません。チェストの横や前にスペースを空けておいてください。
</Warning>

## 買い方・売り方

他のプレイヤーのショップを利用する方法です：

### 購入する
1. ショップの看板を**左クリック**します。
2. チャット欄に「購入したい個数」を入力します。
   - 全て購入したい場合は `all` と入力することもできます。

### 買取ショップへ売る
1. 買取設定（BUYING）になっているショップの看板を**左クリック**します。
2. チャット欄に「売りたい個数」を入力します。

---

## ショップの管理 (コントロールパネル)

自分のショップを**右クリック**すると、管理メニュー（GUI）が開きます。
ここでは以下の設定が可能です：

- **価格変更**: 販売価格を変更します。
- **買い/売りの切り替え**: 「販売（SELLING）」と「買取（BUYING）」を切り替えます。
- **ショップの削除**: ショップを撤去します（チェスト内のアイテムは保持されます）。
- **在庫確認**: 現在の在庫状況を確認したり、補充したりできます。

## 便利なコマンド

| コマンド | 説明 |
| :--- | :--- |
| `/qs find <アイテム名>` | 指定したアイテムを販売している一番近いショップを探します。 |
| `/qs staff <add/del> <名前>` | ショップの管理権限を他のプレイヤーに付与/削除します。 |
| `/qs buy` | 見ているショップを「買取モード」に変更します。 |
| `/qs sell` | 見ているショップを「販売モード」に変更します。 |
| `/qs price <価格>` | 見ているショップの価格を直接変更します。 |
