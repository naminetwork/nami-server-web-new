# NamiServerへようこそ！

NamiServer（ナミサーバー）に参加して、広大な世界での冒険を始めましょう。

## 🎮 機種別の参加手段

Nami Networkは、様々なデバイスから参加可能です。お手持ちの機種を選択して、参加手順を確認してください。

<Tabs items={["PC (Java/Windows版)", "スマホ・タブレット", "家庭用ゲーム機 (Switch/PS系)"]}>
  <Tab value="PC (Java/Windows版)">
    1. Minecraftを起動し、**「マルチプレイ」**を選択します。
       ![メインメニュー](/wiki/join-java-1.png)

    2. **「サーバーを追加」**をクリックします。
    3. 以下を入力し、「完了」を押すとサーバーリストに登録されます。
       - **サーバー名**: `Nami Network` (なんでもok)
       - **サーバーアドレス**: `play.naminetwork.jp`
       ![サーバー追加画面](/wiki/join-java-2.png)

    4. サーバーリストから参加してください。
       ![サーバーリスト](/wiki/join-java-3.png)
  </Tab>

  <Tab value="スマホ・タブレット">
    1. Minecraftを起動し、**「遊ぶ」** > **「サーバー」**タブを選択します。
       ![サーバータブを選択](/wiki/join-mobile-1.png)

    2. リストの一番下にある**「サーバーを追加」**をクリックします。
    3. 以下を入力して保存し、参加してください。
       - **サーバー名**: `Nami Network` (なんでもok)
       - **サーバーアドレス**: `play.naminetwork.jp`
       - **ポート**: `19132`
       ![サーバー情報を入力](/wiki/join-mobile-2.png)
  </Tab>

  <Tab value="家庭用ゲーム機 (Switch/PS系)">
    家庭用ゲーム機では、標準機能で外部サーバーを追加できない制限があります。以下の**「フレンド参加」**手段をご利用ください。

    1. **「遊ぶ」** > **「フレンド」** > **「プレイヤーを検索」**を選択します。
       ![フレンドを追加を選択](/wiki/join-friend-4.png)

    2. プレイヤータグ **`NAMINETWORK1`** を検索し、フレンドに追加します。
       ![プレイヤータグを検索](/wiki/join-friend-2.png)
       ![フレンドに追加済み](/wiki/join-friend-3.png)

    3. 追加後、「参加可能なフレンド」として **「なみ鯖 | NamiServer」** が表示されるので、そこから参加してください。
       ![参加可能なフレンド](/wiki/join-friend-1.png)
       ![フレンドリストから参加](/wiki/join-friend-5.png)

    <Tip>
      もし上記の手順で参加できない場合は、スマホアプリ**「Bedrock Together」**や、DNS設定を変更する手段でも参加可能です。詳しくはDiscordコミュニティで質問してください。
    </Tip>
  </Tab>
</Tabs>

## 遊べるモード
当サーバーには複数の魅力的なワールドがあります。

1. **NamiEarth (地球鯖)**: リアルな地球マップで国を作り、経済や外交を楽しむメインモード。
2. **サバイバル鯖**: 土地保護・Jobs・ショップ・資源ワールドなど、経済要素が充実したサバイバルモード。[マップを見る](https://survivalmap.naminetwork.jp/)
3. **ミニゲーム**: 友達と短時間で盛り上がれるオリジナル競技モード。
