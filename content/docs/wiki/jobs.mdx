# 💼 Jobs ガイド

<Info>
  このシステムは **NamiEarth** と **サバイバル鯖** の両方で利用できます。
</Info>

Jobsは「仕事」に就くことで、普段のプレイがそのまま報酬になるシステムです。

1. `/jobs browse` でジョブ一覧GUIを開く
2. 就きたい仕事をクリック
3. あとは普段通りプレイするだけ！

<Warning>
  同時に就ける仕事は**最大3つ**までです。
</Warning>

---

## 📋 コマンド一覧

| コマンド | 説明 |
| :--- | :--- |
| `/jobs browse` | ジョブ一覧をGUIで表示 |
| `/jobs join <ジョブ名>` | ジョブに就く |
| `/jobs leave <ジョブ名>` | ジョブを辞める |
| `/jobs leaveall` | すべてのジョブを辞める |
| `/jobs info <ジョブ名>` | ジョブの詳細情報を確認 |
| `/jobs stats` | 自分のレベル・経験値を確認 |
| `/jobs top <ジョブ名>` | ランキング表示 |
| `/jobs points` | ポイント確認 |
| `/jobs shop` | ジョブショップを開く |

## ⛏️ 職業一覧

全職業の**最大レベルは200**です。

| 職業 | 説明 | 主なアクション |
| :--- | :--- | :--- |
| 🪓 Woodcutter | 木こり | 各種原木の伐採 |
| ⛏️ Miner | 採掘師 | 鉱石・石の採掘、レール設置 |
| 🏗️ Builder | 建築家 | 各種ブロックの設置 |
| 🔨 Digger | 掘削師 | 土・砂・砂利・粘土の掘削 |
| 🌾 Farmer | 農家 | 作物収穫、繁殖、テイム、羊毛刈り |
| 🎣 Fisherman | 漁師 | 魚釣り |
| 🧪 Brewer | 醸造家 | ポーション醸造 |
| 🔧 Crafter | クラフター | アイテムのクラフト・精錬 |
| ✨ Enchanter | エンチャンター | エンチャント付与 |
| ⚔️ Weaponsmith | 武器鍛冶 | 武器・防具のクラフト・修理・精錬 |

<Tip>
  各職業の詳しい対象アイテムはゲーム内で `/jobs info <ジョブ名>` を使って確認できます。
</Tip>

## 🔄 サーバーごとの違い

| | NamiEarth | サバイバル鯖 |
| :--- | :--- | :--- |
| 報酬の種類 | 経験値 + ポイント | **お金** + 経験値 + ポイント |

---

## 💡 知っておくべきこと

- ジョブを辞めると**レベルが30%失われます**
- ジョブ数が多いほど報酬が少し減少します
- 稼いだポイントで `/jobs shop` からアイテムを購入できます
