# ⚔️ 戦争 (SiegeWar) ガイド

<Info>
  このシステムは**国家鯖**と**NamiEarth**で利用できます。
</Info>

当サーバーでは「SiegeWar」プラグインを使用し、国家間の領土争奪が可能です。無秩序な争いを防ぐため、**「合意制」**かつ**「スケジュール制」**の特別ルールに基づいています。

## 1. 開戦までの流れ
戦争を始めるには、事前の外交と双方の合意が必要です。

1. **合意形成**: Discordやゲーム内で相手国と戦争の合意を取ります。
2. **平和解除**: 双方の町で `/t toggle peaceful` を実行し、平和主義モードを解除します。
3. **バナー準備**: 攻撃側は羊毛と棒で「バナー（旗）」を作成します。
4. **宣戦布告**: **木曜日**に、攻撃側の国王/将軍がバナーを持って相手の町の境界（Wilderness）へ行き地面に設置します。
5. **支払い**: `/sw confirm` で軍資金を支払い、開戦を確定します。

### 💰 戦争費用
戦争は 1 Plot ごとに費用が発生します。
- **手付金**: $5.0 (返金不可の手数料)
- **軍資金**: $25.0 (敗北時に敵に没収、勝利時に返金される賭け金)
**※人口（Townyレベル）が高い町への攻撃はコストが最大+50%増加します。**

---

## 2. 戦争スケジュール
宣戦布告から決着まで、**合計7回の戦闘（バトルセッション）**が行われます。

- **宣戦布告**:| 曜日 | 時間 |
| :--- | :--- |
| **木曜日** | **宣戦布告のみ可能** |
| **土曜日** | 10:00～11:00 / 21:00～22:00 |
| **日曜日** | 10:00～11:00 / 21:00～22:00 |
| **翌週 土曜日** | **10:00 (最終決戦)** |
(ここで勝敗が確定)

---

## 3. 軍事ランクの付与
**重要**: プレイヤーが戦績を上げるには、国王や市長から「軍事ランク」を付与される必要があります。

- **国家ランク**: `/n rank add [名前] [ランク]`
  - `private`, `sergeant`, `lieutenant`, `captain`, `major`, `colonel`（兵士）
  - `general`（将軍：略奪、占領、撤退などの権限を持つ）
- **町ランク**: `/t rank add [名前] guard`（警備兵）

---

## 4. 戦闘のルールと仕様
戦闘エリア（バナー周辺半径300m）では特殊ルールが適用されます。

### ⚔️ PvPとアイテム保護
- **強制PvP**: 町の設定に関わらず、エリア内は強制的に対人攻撃が可能になります。
- ** KeepInventory**: アイテムは落としませんが、死亡時に**装備の耐久値が5%減少**します。
- **チャット制限**: 出会い頭の通信を遮断し、軍事意識を高めるため一部の個人宛メッセージが制限されます。

### 🎯 ポイント獲得アクション
- **バナー占領**: 敵のバナー付近に7分間留まると占領完了。毎分30ポイント加算。
- **敵の撃破**: 公式参加者を倒すと150ポイント獲得。

---

## 5. 勝敗と結果
### 🏆 攻撃側の勝利（ポイントプラス時）
- **略奪 (Plunder)**: 町から $40/plot 分の資金を奪い、国家銀行に入金。
- **占領 (Invade)**: 町を属国化し、自国の一部にする（首都は不可）。

### 🛡️ 防衛側の勝利（ポイントマイナス/0時）
- **防衛成功**: 攻撃側の軍資金を没収し、町の平和が保たれます。

---

## 6. 反乱 (Revolt)
占領された町の市長は、町の境界にバナーを設置することでいつでも「反乱攻城戦」を開始し、独立を目指すことができます。

---

## 7. 重要コマンド一覧
| コマンド | 説明 |
| :--- | :--- |
| `/sw hud [町名]` | 戦況HUDのON/OFF |
| `/sw spawn [町名]` | コマンダーの元へテレポート |
| `/sw collect` | 国家から配分された給料（戦功報酬）を受け取る |
| `/sw surrender` | 防衛側が降伏する |
| `/t toggle peaceful` | 平和主義モードの切り替え |
| `/sw preference [項目]` | UI（ボスバー、ビーコン）の設定 |
