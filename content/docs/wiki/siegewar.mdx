# ⚔️ 戦争 (SiegeWar) ガイド

<Info>
  このシステムは**NamiEarth**で利用できます。
</Info>

当サーバーでは「SiegeWar」プラグインを使用し、国家間の領土争奪が可能です。無秩序な争いを防ぐため、**「スケジュール制」**の特別ルールに基づいています。**平和主義（Peaceful）モードを解除（OFF）している国は、合意なく宣戦布告の対象となります。**

## 1. 開戦までの流れ
戦争を始めるには、事前の外交と双方の合意が必要です。

1. **ターゲット選定**: 平和主義モードを解除（OFF）している町を確認します。
2. **バナー準備**: 攻撃側は羊毛と棒で「バナー（旗）」を作成します。
3. **宣戦布告**: **木曜日**に、攻撃側の国王/将軍がバナーを持って相手の町の境界（Wilderness）へ行き地面に設置します。
4. **支払い**: `/sw confirm` で軍資金を支払い、開戦を確定します。

### 💰 戦争費用
戦争は 1 Plot ごとに費用が発生し、町の規模や場所によって変動します。
- **手付金 (Upfront)**: $4.0 / Plot (返金不可の手数料)
- **軍資金 (Warchest)**: $10.0 / Plot (敗北時に敵に没収、勝利時に返金される預け金)
**※首都への攻撃はコストが+30%増加します。**
**※人口（Townyレベル）によってもコストが別途変動します。**

---

## 2. 戦争スケジュール
宣戦布告から決着まで、**合計7回の戦闘（バトルセッション）**が行われます。

- **宣戦布告**: **木曜日のみ可能**
- **戦闘期間**: 土曜日〜翌週土曜日にかけて**合計7セッション**行われます。

| 曜日 | 時間 | 内容 |
| :--- | :--- | :--- |
| **木曜日** | - | **宣戦布告のみ可能** |
| **土曜日** | 10:00～11:00 / 21:00～22:00 | 第1・2セッション |
| **日曜日** | 10:00～11:00 / 21:00～22:00 | 第3・4セッション |
| **翌週 土曜** | 10:00 / 21:00 | 第5・6セッション |
| **翌週 日曜** | 10:00 | 最終セッション・結果確定 |


---

## 3. 軍事ランクの付与
**重要**: プレイヤーが戦績を上げるには、国王や市長から「軍事ランク」を付与される必要があります。

- **国家ランク**: `/n rank add [名前] [ランク]`
  - `private`, `sergeant`, `lieutenant`, `captain`, `major`, `colonel`（兵士）
  - `general`（将軍：略奪、占領、撤退などの権限を持つ）
- **町ランク**: `/t rank add [名前] guard`（警備兵）

---

## 4. 戦闘のルールと仕様
### ⚔️ PvPと仕様
- **強制PvP**: 町の設定に関わらず、エリア内（半径200ブロック以内）は強制的に対人攻撃が可能になります。
- **デスペナルティ**: アイテムロストについてはサーバーの基本設定に従いますが、死亡時に**装備の耐久値が5%減少**します。
- **建築・破壊制限**: 戦争エリア内（Wilderness）では、**黒曜石、氷、溶岩/水バケツ、TNT**の設置や使用が制限されます。

### 🎯 ポイント獲得アクション
勝敗を分けるポイント獲得方法は主に2つです。

- **バナー占領 (Banner Control)**:
    - 敵のバナー付近に **7分間** 留まると占領完了。
    - 占領中は **20秒ごとに34ポイント**（1分間で102ポイント）が加算されます。
    - **逆転ボーナス**: 相手チームからコントロールを奪い返すと、ポイント加算率が **3倍** にアップします。
- **敵の撃破 (Player Kills)**:
    - 軍事ランクを持つ公式参加者を倒すと **150ポイント** 獲得。

**※ポイント上限**: 1回の戦争での累計ポイントは **±20,000** で頭打ちとなります。

## 5. 視覚補助とインジケーター
戦況を把握するための便利な機能が用意されています。

- **ビーコン (Beacon Markers)**:
    - ビーコンの色でバナーの状態がわかります。
        - <span style={{color: 'yellow'}}>●</span> **黄**: あなたが現在バナーを捕獲（Cap）中です。
        - <span style={{color: 'green'}}>●</span> **緑**: あなたのチームがバナーを支配しています。
        - <span style={{color: 'red'}}>●</span> **赤**: 敵チームがバナーを支配しています。
- **HUDとボスバー**:
    - 画面上部のボスバーで、現在のバナー捕獲の進行状況がリアルタイムで表示されます。
    - `/sw hud [町名]` で詳細な戦況を画面に表示できます。

---

## 6. 勝敗と勝利の種類
7セッション終了時の最終ポイント（Siege Balance）で勝敗が決まります。点差によって勝利の重みが変わります。

| 勝利名 | 条件 | 結果 |
| :--- | :--- | :--- |
| **圧倒的勝利 (Crushing)** | 15,000点以上 | 攻撃側は「占領」および「略奪」が可能、軍資金獲得。 |
| **決定的勝利 (Decisive)** | 5,000点以上 | 攻撃側は「占領」または「略奪」が可能、軍資金獲得。 |
| **僅差の勝利 (Close)** | 1点以上 | 攻撃側は「略奪」のみ可能、軍資金獲得。 |
| **防衛成功** | 0点以下 | 防衛側が攻撃側の軍資金を全額没収。 |


---

## 7. 反乱 (Revolt)
占領された町の市長は、占領から**72時間（3日間）**経過後、町の境界にバナーを設置することで「反乱攻城戦」を開始し、独立を目指すことができます。

**※攻撃免除期間 (Siege Immunity):**
- 新しい町、または戦争が終了したばかりの町は、**168時間（7日間）**の攻撃免除期間が与えられます（首都の場合は14日間）。

---

## 8. 戦略アドバイス
- **バトルコマンダーの活用**: 国王や将軍が前線（SiegeZone）に生存している場合、`/sw spawn [町名]` で軍のメンバーを即座にコマンダーの元へテレポートさせることができます。
- **逆転ボーナスの狙い目**: ポイントが大きく離されていても、バナーを奪い返せば3倍速でポイントを稼げるため、少数精鋭での奇襲やバナー奪還は非常に有効な戦術です。
- **装備の準備**: 本サーバーでは戦争中も**アイテムロストが発生します**。予備の装備や物資を十分に用意して挑みましょう。

---

## 9. 重要コマンド一覧
| コマンド | 説明 |
| :--- | :--- |
| `/sw hud [町名]` | 戦況HUDのON/OFF |
| `/sw spawn [町名]` | 生存しているバトルコマンダー（国王/将軍）の元へテレポート |
| `/sw collect` | 国家から配分された給料（戦功報酬）を受け取る |
| `/sw surrender` | 防衛側が降伏する |
| `/t toggle peaceful` | 平和主義モードの切り替え |
| `/sw preference [項目]` | UI（ボスバー、ビーコン）の設定 |
