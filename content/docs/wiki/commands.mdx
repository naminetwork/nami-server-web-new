# 🏰 サーバー総合コマンドガイド

<Info>
  **参加しているサーバー**（サバイバル鯖・NamiEarthなど）によって対応していないコマンドがあります。
</Info>

サバイバル生活をより便利にするための基本的なコマンドをまとめています。
テレポート、拠点設定、さらにはNamiEarth（Towny）での操作まで網羅しています！

## 1. 🏃 TPA（テレポート申請）
他のプレイヤーの場所へ移動したり、移動を許可したりするコマンドです。

| コマンド | 説明 |
| :--- | :--- |
| `/tpa [プレイヤー名]` | 指定したプレイヤーにテレポート申請を送ります。 |
| `/tpahere [プレイヤー名]` | 指定したプレイヤーを自分の元へ呼ぶ申請を送ります。 |
| `/tpaccept` | 届いているテレポート申請を**許可**します。 |
| `/tpdeny` | 届いているテレポート申請を**拒否**します。 |

---

## 2. 🏠 ホーム（拠点）の設定
お気に入りの場所を保存して、いつでも戻ってこれるようにします。

| コマンド | 説明 |
| :--- | :--- |
| `/sethome [名前]` | 今立っている場所を「ホーム」として保存します。 |
| `/home [名前]` | 保存したホームへテレポートします。 |
| `/edithome [名前]` | 保存したホームの名前変更や削除をすることができます。 |
| `/homes` | 自分が設定しているホームの一覧を確認します。 |

---

## 3. 🌍 共通・初期地点への移動
ロビーや各サーバーのメイン拠点へ戻るためのコマンドです。

| コマンド | 説明 |
| :--- | :--- |
| `/spawn` | 今いるサーバーの初期スポーン地点へ戻ります。 |
| `/hub` | サーバーネットワークのメインロビーへ戻ります。 |
| `/back` | テレポートや死亡する直前の場所へ戻ります。 |

---

## 4. 🚀 サーバー間の移動
別のサーバー（サバイバル鯖からNamiEarthなど）へ直接移動するコマンドです。

| コマンド | 説明 |
| :--- | :--- |
| `/server [サーバー名]` | 指定した名前のサーバーへ移動します。 |

---

## 5. 🚩 土地の保護（PlayerGuard）
主にサバイバルサーバーで、自分の土地を範囲指定して守る機能です。

| コマンド | 説明 |
| :--- | :--- |
| `/claim` | 金の斧で選択した範囲を自分の領地として保護します。 |
| `/disclaim` | 立っている場所の保護を解除します。 |
| `/pg add [プレイヤー名]` | 領地にメンバーを追加して建築を許可します。 |

<Tip>
  さらに詳しい使い方は [PlayerGuard Wiki](/wiki/playerguard) をご覧ください。
</Tip>

---

## 6. 🏘️ 建国プラグイン（Towny）基本操作
NamiEarthで町に参加したり、町の情報を確認したりするコマンドです。

| コマンド | 説明 |
| :--- | :--- |
| `/t` | 自分の所属している町の情報を表示します。 |
| `/t spawn` | 町のスポーン地点（広場）へテレポートします。 |
| `/t deposit [金額]` | 町の銀行にお金を寄付します（維持費に必要です）。 |
| `/tc` | 町チャットに切り替えます（住民同士の会話）。 |

<Tip>
  さらに詳しい使い方は [Towny Wiki](/wiki/towny) をご覧ください。
</Tip>

---

### 💡 ヒント
* `/tpa` などのテレポートは、移動中にダメージを受けたり動いたりするとキャンセルされる場合があります。
* 町や土地の維持費、保護ルールは各Wikiをよく読んで確認しましょう！
