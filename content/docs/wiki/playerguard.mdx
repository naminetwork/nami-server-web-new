# 🏰 土地保護ガイド

<Info>
  このシステムは**サバイバル鯖専用**です。NamiEarthではTownyを使用してください。
</Info>

PlayerGuardには2つの保護システムがあります：
1. **無料クレーム** - 金の斧で自由に土地を保護
2. **プロット購入** - 住宅街の区画を購入

---

## 1. 🚩 無料クレームシステム

### 使い方

1. **金の斧**を手に持つ
2. 保護したい範囲の**1点目を右クリック**
3. **2点目を右クリック**
4. `/claim` で保護確定

### コマンド

| コマンド | 説明 |
|---------|------|
| `/claim` | 選択範囲を保護 |
| `/cancel` | 選択をキャンセル |
| `/disclaim` | 保護を解除 |
| `/flags` | 権限管理GUIを開く |

---

## 2. 🏘️ プロット購入システム

住宅街の区画を購入できます。中央に近いほど高額です。

| コマンド | 説明 |
|---------|------|
| `/plot buy` | 立っているプロットを購入 |
| `/plot info` | プロットの情報表示 |
| `/plot list` | 所有プロット一覧 |
| `/plot add <名前>` | メンバーを追加 |
| `/plot remove <名前>` | メンバーを削除 |
| `/plot flags` | フラグ管理GUI |
| `/plot release` | プロットを手放す |

<Warning>
  **30日間ログインしないとプロットは自動解放されます。**
</Warning>

---

## 3. 👥 メンバー管理

自分以外のプレイヤーと一緒に作業したい場合に使用します。

- `/pg add <名前>` : プレイヤーをメンバーに追加（建築や破壊が可能になります）。
- `/pg remove <名前>` : メンバーから削除し、権限を剥奪します。

---

## 4. 📋 領地の情報確認・譲渡

- `/pg info` : 領地の所有者、座標、メンバー一覧、サイズを表示。
- `/pg transfer <名前>` : オーナー権を別のプレイヤーに譲渡。

---

## 5. 📈 保護できる範囲について

当サーバーでは、プレイヤーの活動量に応じて保護枠が拡大します。

- **プレイ時間連動**: サーバーで遊んでいる時間が長くなるほど、保護できる合計ブロック数（上限値）が増えていきます。
- **枠の拡大**: 最初は小さな家から始め、長く遊ぶことでより大きな拠点や街を保護できるようになります。

---

## 6. 🧳 チェスト共有に関する注意

PlayerGuardでメンバーを追加しただけでは、チェストが開けない場合があります。その際はTownyの権限設定などを確認してください。

| コマンド | 手順・説明 |
|---------|-----------|
| `/lock <名前>` | コマンドを打った後、共有したいチェストを**左クリック**。 |
| `/unlock` | コマンドを打った後、解除したい自分のチェストを**左クリック**。 |

---

## 💡 ヒント

- `/playerguard` は `/pg` で略せます
- 金の斧を持って空クリックすると境界線が表示されます
