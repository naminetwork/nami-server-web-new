# 🧪 ポーション作成ガイド

<Warning>
  **採掘速度上昇VII**など一部の高レベルポーションは**国家鯖専用**です。
</Warning>

MCMMOの「錬金術」スキルを上げることで、バニラでは作れない特殊なポーションが作成可能になります。

## 🎯 ポーション一覧（レベル別）

| ポーション | 材料 | レベル | 効果 |

| **奇妙なポーション** | 水入り瓶 + ネザーウォート | 0 | 全ての基本 |

| **耐火** | 奇妙 + マグマクリーム | 0 | 火災ダメージ無効 |
| **俊敏** | 奇妙 + 砂糖 | 0 | 移動速度上昇 |
| **水中呼吸** | 奇妙 + フグ | 0 | 水中で息ができる |
| **治癒** | 奇妙 + きらめくスイカ | 0 | 即時回復 |
| **力** | 奇妙 + ブレイズパウダー | 0 | 攻撃力上昇 |

## 🎯 まとめ

### 最初の目標
1. **レベル100**：採掘速度上昇II（石即掘り）
2. **レベル400**：採掘速度上昇VII（深層岩即掘り）

採掘速度上昇IIを手に入れると、ブランチマイニングの効率が劇的に向上します。まずはレベル100を目指して醸造を繰り返しましょう！⛏️✨
