# ⛏️ サバイバル鯖ガイド

サバイバル鯖は、土地保護・Jobs・ショップ・資源ワールドなど経済要素が充実したサバイバルモードです。

---

## 🗺️ ワールド構成

| ワールド | 移動コマンド | 説明 |
| :--- | :--- | :--- |
| メインワールド | `/spawn` | メインの活動拠点 |
| 旧スポーン | `/spawn mae` | 旧スポーン地点 |
| 資源ワールド | `/spwn shigen_world` | 定期リセットされる採取用ワールド |
| 資源ネザー | `/spwn shigen_nether` | 定期リセットされるネザー |
| 資源エンド | `/spwn shigen_end` | 定期リセットされるエンド |

<Info>
  資源ワールドは定期的にリセットされます。大切なアイテムは必ず持ち帰りましょう。
</Info>
