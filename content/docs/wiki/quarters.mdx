# 🏘️ Quarters 3D区画使い方ガイド

<Info>
  このシステムは**NamiEarth**で利用できます。
</Info>

Quartersは、Townyの土地システムを拡張した**3D区画システム**です。
通常のPlotはチャンク単位の平面管理ですが、Quartersでは**立体的で自由な形の区画（アパートや部屋単位）**を作成・管理できます。

## 通常のPlotとの違い
- **Plot**: 16x16ブロックのチャンク単位。平面的で固定サイズ。
- **Quarter**: 自由な大きさの**立方体（3D）**を組み合わせて作成可能。マンションの1室や、建物の階層ごとに所有者を分けられます。

---

## 2. 区画の作成（市長・助役向け）
区画を作成できるのは**市長**または**助役（Assistant）**だけです。

### 📌 基本的な作成手順
1. **ワンド入手**: 火打石（Flint）がワンドです。`/q wand` で入手可能。
2. **範囲選択**: 火打石を持ち、区画にしたい場所の対角線上の2つの角を右クリック/左クリックで選択します。
3. **作成**: `/q create` で選択範囲が区画になります。
4. **販売**: 作成した区画の中に立ち、`/q sell <金額>` で売りに出します。

---

## 3. 区画の購入・所有
- **購入**: 販売中の区画の中に立ち、`/q claim` を実行。
- **販売**: 自分が所有している区画を他人に売りたい場合は `/q sell <金額>`。
- **放棄**: `/q unclaim` で所有権を放棄。
- **確認**: `/q here` で現在の場所の区画情報を表示。

---

## 4. 権限管理とトラスト

### 🤝 トラスト（共同建築）
友達と一緒に住んだり作業したりしたい場合に使います。
- `/q trust add <名前>` : プレイヤーに全権限を共有。
- `/q trust remove <名前>` : トラストを解除。

### 🔒 詳細な権限設定
`/q set perm <対象> <行動> <on/off>`
- **対象**: `resident` (自分), `friend` (フレンド), `ally` (同盟), `outsider` (他人)
- **行動**: `build` (設置), `destroy` (破壊), `switch` (操作), `itemuse` (使用)

---

## 5. 特別区画タイプ
市長・助役は `/q set type <タイプ>` で区画の性質を変更できます。

| タイプ | 特徴 | 用途 |
| :--- | :--- | :--- |
| **Apartment** | 標準の区画。 | 一般住居 |
| **Inn (宿屋)** | **ベッドが使用可能**になります。 | ホテル・宿 |
| **Station (駅)** | トロッコやボートが使用可能です。 | 駅・港 |

<Tip>
  `/q toggle embassy` をONにすると、**町の外の人（旅行者）**でも宿屋や駅などの機能を利用できるようになります。
</Tip>

---

## 6. コマンド一覧
| コマンド | 説明 |
| :--- | :--- |
| `/q wand` | 範囲選択用の火打石を入手 |
| `/q here` | 立っている区画の情報を表示 |
| `/q claim` | 区画を購入 |
| `/q trust add [名前]` | 建築メンバーを追加 |
| `/q set type [タイプ]` | 区画の属性を変更 (市長のみ) |
| `/q sell [金額]` | 区画を販売 (市長のみ) |
| `/q set name [名]` | 区画に名前を付ける |
| `/q toggle constantoutlines` | 境界線を常に表示 |
| `/q delete` | 区画を削除 (市長のみ) |

---
### 💡 活用例：マンション
市長が建物の各階や各部屋を別々の Quarter として作成し、格安で販売することで、1つのチャンク内に多くの住民が住める集合住宅を作ることができます。
