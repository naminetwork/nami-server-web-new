# 🛠️ カスタムレシピ

サーバー独自のクラフトレシピを紹介します。
これらのレシピを使用することで、通常は入手が困難なアイテムや、別の方法で作成できるアイテムを作成できます。

---

## 🛤️ 加速レール (コスト軽減)
通常は金が必要な加速レールを、**鉄インゴット**を使用して作成できます。

![加速レールのレシピ](/wiki/recipe-powered-rail.png)

| 材料 | 個数 |
| :--- | :--- |
| 鉄インゴット | 6 |
| 棒 | 1 |
| レッドストーンダスト | 1 |
| 金インゴット | 1 |
| **完成品** | **加速レール x8** |

---

## 🏜️ エンドストーン
丸石と砂を組み合わせることで、エンドストーンを作成できます。

![エンドストーンのレシピ](/wiki/recipe-end-stone.png)

| 材料 | 個数 |
| :--- | :--- |
| 丸石 | 5 |
| 砂 | 4 |
| **完成品** | **エンドストーン x8** |

---

## 🧪 ドラゴンブレス
コーラスフルーツとガラス瓶を使用して、ドラゴンブレスを作成できます。

![ドラゴンブレスのレシピ](/wiki/recipe-dragons-breath.png)

| 材料 | 個数 |
| :--- | :--- |
| コーラスフルーツ | 4 |
| ガラス瓶 | 1 |
| **完成品** | **ドラゴンブレス x1** |

---

## 🔮 エンドポータルフレーム
エンドポータルを構成するフレーム自体を作成できます。非常に強力なアイテムです。

![エンドポータルフレームのレシピ](/wiki/recipe-end-portal-frame.png)

| 材料 | 個数 |
| :--- | :--- |
| コーラスフルーツ | 4 |
| エンドストーン | 4 |
| エンダーアイ | 1 |
| **完成品** | **エンドポータルフレーム x4** |

---

<Note>
  レシピが正しく動作しない場合は、材料の配置が画像と同じか再度ご確認ください。
</Note>
