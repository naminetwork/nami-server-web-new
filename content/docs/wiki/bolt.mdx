# 🔒 鍵かけ・ブロック保護 (Bolt) 使い方ガイド

<Warning>
  このシステムは**国家鯖専用**です。サバイバル鯖ではPlayerGuardを使用してください。
</Warning>

Boltは、チェスト、ドア、かまどなどのブロックを個別に保護するためのプラグインです。土地保護（Towny）とは別に、中身やアクセス権を細かく管理したい場合に非常に便利です。

> **基本機能**: チェストやドアを設置すると、自動的にあなたの「プライベート（自分専用）」として保護されます。

## 1. 🔒 保護の基本設定
設置したブロックを手動でロック、または解除するコマンドです。

| コマンド | 説明 |
| :--- | :--- |
| `/bolt lock` | ブロックを左クリックして、自分専用に保護。 |
| `/bolt unlock` | ブロックを左クリックして、保護を完全に解除。 |

---

## 2. 👥 権限の変更・共有 (Modify)
友達と一緒にチェストを使いたい場合などは、このコマンドで「権限」を編集します。

| コマンド | 説明 |
| :--- | :--- |
| `/bolt modify [名前] player.full [true/false]` | 対象を左クリック。指定プレイヤーの全権限（開ける・壊す）を設定。 |
| `/bolt modify [名前] group.full [true/false]` | 対象を左クリック。指定グループの権限を設定。 |

**よく使う例:**
- **友達を追加する**: `/bolt modify <名前> player.full true`
- **権限を消す**: `/bolt modify <名前> player.full false`

---

## 3. ⚙️ 保護タイプの変更
用途に合わせて保護タイプを変更できます。コマンド入力後にブロックを左クリックしてください。

| コマンド | タイプ | 特徴 |
| :--- | :--- | :--- |
| `/bolt set type private` | **プライベート** | あなたと追加したメンバーだけが使用可能です。 |
| `/bolt set type public` | **パブリック** | 誰でも使えますが、保護（所有権）自体は残ります。 |
| `/bolt set type donation` | **ドネーション** | **投函のみ可能**。取り出しは所有者だけができます。 |

---

## 4. 📋 情報の確認と所有権の譲渡
| コマンド | 説明 |
| :--- | :--- |
| `/bolt info` | ブロックを左クリック。所有者や設定されている権限一覧を表示。 |
| `/bolt transfer [名前]` | ブロックを左クリック。保護の所有権を相手に譲ります。 |

---

## 5. ⚡ 便利な一括操作モード
たくさんのチェストを一度に処理したい時に便利な「永続モード」です。

- `/bolt mode persist` : **超便利。** このモードをONにすると、一度コマンドを打つだけで、複数のブロックを連続でクリックして同じ操作（保護や編集）ができます。再度打つと解除されます。

---
<Tip>
  `modify` コマンドは少し複雑に見えますが、基本的には「誰に（プレイヤー名）」「どの権限を（player.full）」「有効(true)にするか」を指定する流れになります。
</Tip>
