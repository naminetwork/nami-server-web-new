# 🏰 Towny 市長＆プレイヤーガイド

<Info>
  このシステムは**NamiEarth**で利用できます。
</Info>

Townyは、Minecraftの広大な土地で「町」や「国」を作り、住民同士で協力しながら領土を広げていくためのシステムです。

## 1. 基本・共通コマンド
誰でも使える、情報の確認やマップ表示に関するコマンドです。

| コマンド | 説明 |
| :--- | :--- |
| `/towny map` | **最重要。** チャット欄に近隣のマップを表示します。 `+`は現在地、`-`は土地、`$`は販売中の土地。 |
| `/towny prices` | 町の作成、維持費、土地の請求費用などの価格一覧を表示。 |
| `/towny top land` | 土地を多く持っている町のランキングを表示。 |
| `/res` | 自分のステータス（所属、所持金、ランクなど）を表示。 |
| `/res friend add [名前]` | 他プレイヤーをフレンドに追加（土地の権限設定で使用）。 |
| `/res set mode plotborder` | **便利。** 土地の境界線をパーティクルで可視化します。 |

## 2. 町（Town）のコマンド
町に所属することで、土地の保護や、町専用のスキル、チャットが利用可能になります。

### 一般住民向け
| コマンド | 説明 |
| :--- | :--- |
| `/t` | 自分の所属している町の詳細情報を表示。 |
| `/t list` | サーバーにあるすべての町をリスト表示。 |
| `/t spawn` | 町のスポーン地点（広場）へテレポート。 |
| `/t join [町名]` | 招待されている町に参加。 |
| `/t leave` | 現在の町から脱退。 |
| `/t deposit [金額]` | 町の銀行にお金を寄付（維持費の支払いに必要）。 |

### 市長・管理職向け
**市長**や、市長から権限を与えられた**助役（Assistant）**が使用します。

#### 🏠 町の作成と運営
- `/t new [町名]` : 新しい町を作成します（銀行残高から費用が引かれます）。
- `/t delete` : 町を削除します。

#### 🗺️ 領土の拡大
- `/t claim` : 立っているチャンクを町の領土として請求。隣接している必要があります。
- `/t claim outpost` : 本拠地から離れた場所に「飛び地」を作成します。
- `/t unclaim` : 立っている場所の領土を手放します。

#### 👥 住民の管理
- `/t invite [名前]` : プレイヤーを招待。
- `/t kick [名前]` : 住民を追放。
- `/t rank add [名前] [ランク]` : 役職（Assistant, Helper, Sheriff, VIP等）を付与。
- `/t trust add [名前]` : **町全体のすべての土地**での建築権限を与える強力なコマンド。

#### ⚙️ 町の高度な設定
- `/t set taxes [金額]` : 住民から毎日徴収する税金を設定。
- `/t toggle public` : `off`にすると部外者が `/t spawn` を使えなくなります。
- `/t toggle explosion/fire/mobs` : 町全体での爆発・延焼・モブ湧きの切り替え。

## 3. 土地（Plot）の管理
町の中の各チャンク（Plot）を個別に管理します。

### トラスト（権限の共有）
特定の土地（自分の家など）を友達と一緒に使いたい場合に使用します。
- `/plot trust add [名前]` : 指定したプレイヤーに建築・チェスト使用権限を共有。
- `/plot trust remove [名前]` : 共有を解除。

### 土地の売買・種類
- `/plot forsale [金額]` : 自分の土地を売りに出す。住民が購入できるようになります。
- `/plot claim` : 売りに出された土地を自分の所有地にする。
- `/plot set embassy` : **大使館**設定。他の町の人でも購入可能。
- `/plot set shop` : ショップ用。

### 🔒 詳細な権限設定 (Perm)
`/plot set perm [対象] [行動] [on/off]` で細かく制御します。
- **対象**: `resident` (自分), `friend` (フレンド), `ally` (同盟), `outsider` (他人)
- **行動**: `build`, `destroy`, `switch`, `itemuse`

## 4. 国（Nation）のコマンド
複数の町が集まり「国」となることで、同盟や大規模な戦争が可能になります。

| コマンド | 説明 |
| :--- | :--- |
| `/n` | 所属している国の情報。 |
| `/n spawn` | 国の首都（中心地）へテレポート。 |
| `/n new [国名]` | 新しく国を建国（市長のみ実行可）。 |
| `/n invite [町名]` | 他の町を自国に招待。 |
| `/n ally add [国名]` | 他の国を同盟国に追加。 |

---

## 5. チャット機能
Towny専用のクローズドなチャットチャンネルです。

- `/tc` : **町チャット**。同じ町の住民にだけ届きます。
- `/nc` : **国チャット**。同じ国のメンバー全員に届きます。
- `/lc` : **ローカルチャット**。近く（通常100m以内）にいる人にだけ届きます。
- `/g` : **グローバルチャット**。全プレイヤーに届く通常チャットに戻ります。

---
<Tip>
  町の維持には毎日「維持費（Upkeep）」が発生します。 `/t` で残高と維持費を確認し、不足しないよう `/t deposit` しましょう！
</Tip>
