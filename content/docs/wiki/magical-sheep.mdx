# 🐑 MagicalSheep ガイド

<Info>
  このミニゲームは**全サーバー**で遊べます。
</Info>

MagicalSheepは、様々な特殊能力を持つ「羊」を武器にして戦うPVPゲームです。ステージ上のアイテムを戦略的に使い分けることが勝利の鍵となります。

## 🎯 ゲームの流れ
1. **アイテム収集**: ステージ内のチェストから色とりどりの羊やアイテムを集めます。
2. **羊の発射**: 羊を手に持ち、「**アイテムを捨てる操作（Qキーなど）**」で相手に向けて放ちます。
3. **能力発動**: 羊が敵や障害物に当たると、爆発や毒などの特殊効果が発生します。
4. **生き残り**: 相手をすべて倒し、最後まで生き残れば勝利です。

---

## 📋 操作とテクニック

### 🐑 羊の発射
インベントリから羊を選び、アイテム投棄アクションで攻撃します。羊によって飛距離や弾道が異なります。

### 🏹 矢での起爆（高度な技）
飛んでいる最中の羊に**弓矢**を当てると、その場で能力を強制発動させることができます。タイミングを合わせて空中で爆発させるなどのトリッキーな戦術が可能です。

### ⚡ スタミナとアイテム
剣などの直接攻撃アイテムも含まれていますが、メインは羊の投擲です。人数に応じて入手できる羊の数が調整されます。

---

## 🏆 勝利条件
| モード | 勝利条件 |
| :--- | :--- |
| **チーム戦** | 相手チームのメンバーを全員倒したチームの勝利 |
| **ソロモード** | 最後の1人になるまで戦い抜いた人の勝利 |

---

## ⚠️ 重要なポイント
- **能力の理解**: 羊の色ごとに爆発・毒・移動補助など効果が全く異なります。
- **地形の利用**: ステージの構造を活かして、死角から羊を投げ込みましょう。
- **人数調整**: 参加人数が少ない場合、羊の補充速度や初期数が調整されます。
