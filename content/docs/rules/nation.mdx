# 国家鯖専用ルール

<Info>
  本規約は、Towny機能を利用した国家サーバー（以下「国家」）を利用する際の特別ルールです。
</Info>

国家サーバーに参加した時点で、本規約および **NamiServer 利用規約**のすべてに同意したものとみなされます。両者に矛盾がある場合は、基本規約が優先されます。

## 1. アカウント制限

<Warning>
  **1人につき最大2アカウントまで**厳守。
</Warning>

- サブアカウントを利用してゲームを不当に有利に進める行為（Vote報酬の二重取り、メインとサブでの並行作業等）は禁止です。

## 2. 国名・町名について
- **日本語禁止**: 国名・町名は**英数字（A-Z, 0-9）のみ**使用可能です。
- **不適切な名前**: 管理者が不適切と判断した名前は、強制変更の対象となります。

## 3. 建築・設置に関する規定
- **不正建築**: 浮遊ブロック（本来設置できない場所への設置）やサーバーに負荷をかける建築は撤去されます。
- **攻城戦エリア (SiegeZone)**:
  - 黒曜石、氷、水/溶岩バケツ等の設置制限をグリッチで回避する行為は禁止です。
  - 地下深くへの引きこもりや、上空への要塞建築は制限されます。

## 4. PvPおよび対人行為
- **SiegeZone内は常時PvP有効**: 戦争エリア内では、町のPvP設定に関わらず攻撃が当たります。
- **アイテム保護**: SiegeZone内での死亡時はアイテムロストしません（KeepInventory）。ただし、装備の耐久値が減少します。
- **禁止行為**:
  - 執拗なリスキル（リスポーン地点での待ち伏せ）。
  - 死体撃ちや屈伸煽り等のマナー違反。

## 5. 土地確保と妨害
- **囲い込み禁止**: 他国の周囲を意図的に囲んで拡張を妨害する行為は禁止です。
- **乗っ取り禁止**: 既存の建築物の周囲を後から保護（Claim）して奪い取る行為は禁止です。
- **嫌がらせ設置**: 他国の隣接地に看板やチェストを大量に並べて通行や拡張を妨害する行為は禁止です。

## 6. 戦争 (SiegeWar)
- **合意制**: 各町は初期状態で「平和主義（Peaceful）」モードです。双方合意の上で解除して戦争を行います。合意なく無理に解除を迫る行為は禁止です。
- **スケジュール**:
  - 宣戦布告：**木曜日**のみ
  - バトル：**土・日**の **10:00～11:00 / 21:00～22:00**
- **禁止事項**: スパイ行為（サブ垢での潜入・内部破壊）や、意図的な遅延行為。

## 7. 不当な勧誘行為
- 重要な情報（税金、PvP設定、追放条件など）を意図的に隠したり、誤解を招く説明で国家へ勧誘する行為を禁止します。

## 8. 処分内容
規約違反が確認された場合、以下の処分が行われます。
- 警告 (Warn) / キック (Kick)
- ミュート (Mute) / アクセス禁止 (BAN / IP BAN)
